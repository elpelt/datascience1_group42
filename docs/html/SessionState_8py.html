Usage

import SessionState

session_state = SessionState.get(user_name='', favorite_color='black') session_state.user_name

''

session_state.user_name = 'Mary' session_state.favorite_color

'black'

Since you set user_name above, next time your script runs this will be the result:

session_state = get(user_name='', favorite_color='black') session_state.user_name

'Mary'
